open-hue
===============

[![Build Status](https://travis-ci.com/c-bandy/open-hue.svg?branch=master)](https://travis-ci.com/c-bandy/open-hue)

an open source, web based clone of '[i love hue](http://www.i-love-hue.com/)'.

prerequisites
--------

* node >=8.10

quick start
--------

```bash
# install dependencies
npm install

# local development
npm start
```
